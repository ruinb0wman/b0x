# B0x

一个electron+react的变成辅助工具

## Terminal

* 有多Tab, 能切换多个终端
* 有分屏功能, 某个Tab中可以分屏创建多个终端
* 支持选择shell类型, 例如Powershell, WSL
* 支持管理ssh, 方便连接远程服务器

## AI Chat

* 通过快捷键唤起一个窗口进行AI会话
* 可以自定义使用的模型, 和api密钥
* 支持openai形式的会话

## system tray board

* 支持点击system tray图标显示一个board
* board 支持pomo计时功能
* board 支持历史剪切板功能
* board 支持快捷键唤醒
